# 记账App (Accounting App)

一个基于React Native的移动端记账应用，支持手动记账和OCR截图自动记账功能。

## 功能特性
- 收入/支出记录管理
- 交易分类统计
- 收据截图自动识别记账
- 本地数据备份恢复
